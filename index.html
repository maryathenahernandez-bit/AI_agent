01 WHAT IS AN AGENT?
An agent is anything that can be viewed as perceiving its environment through sensors and acting upon that environment through actuators.
The Agent Cycle
PERCEIVE
via Sensors
→
THINK
Process
→

Note: Every agent can perceive its own actions — but not always the effects of those actions.
02 INTELLIGENT AGENTS (IA)
An Intelligent Agent (IA) is an entity that makes decisions, enabling AI to be put into action. It conducts operations in the place of users or programs after sensing the environment, using actuators to initiate action.
Rational Agent — acts with the best outcome after considering past and current percepts (agent's perceptual inputs at a given instance). AI = study of rational agents.
AI System = Agent + Environment

An AI system is composed of an agent and its environment. The agents act in their environment. The environment may contain other agents. An intelligent agent may learn from the environment to achieve goals.

Example of a simple intelligent agent: a thermostat

04 SENSORS · ACTUATORS · EFFECTORS
📡 Sensor

Detects changes in the environment and sends information to other electronic devices. An agent observes its environment through sensors.

⚙️ Actuator

Component of machines that converts energy into motion. Only responsible for moving and controlling a system. Examples: electric motor, gears, rails.

🦾 Effector

Devices that directly affect the environment. Examples: legs, wheels, arms, fingers, wings, fins, display screens.

06 5 TYPES OF AI AGENTS
⚡
Simple Reflex Agent
SIMPLEST

Decides based only on current percepts. Ignores percept history. Only works in fully observable environments.


❌ PROBLEMS:
  • Very limited intelligence
  • No knowledge of non-perceptual state parts
  • Mostly too big to generate/store
  • Not adaptive to changes
🗺️
Model-Based Reflex Agent
PARTIAL ENV.

Can work in partially observable environments and track situations.


TWO KEY FACTORS:
  • Model — knowledge of how things happen in the world
  • Internal State — current state based on percept history

Updates require: how the world evolves + how actions affect the world.

🎯
Goal-Based Agent
PROACTIVE

Expands model-based agent by having goal information. Chooses actions to achieve goals. May consider long action sequences.


Consideration of scenarios = searching and planning → makes agent proactive.

📊
Utility-Based Agent
BEST ACTION

Like goal-based + extra utility measurement. Acts on goals AND the best way to achieve them.


Useful when multiple alternatives exist. Utility function maps each state to a real number to check efficiency.

🧬
Learning Agent
MOST ADVANCED · SELF-IMPROVING

Can learn from past experiences. Starts with basic knowledge and adapts automatically through learning. Able to learn, analyze performance, and find new ways to improve.

📚 Learning Element
Responsible for making improvements by learning from the environment.
🔍 Critic
Provides feedback to the learning element on how well the agent is doing vs. a fixed performance standard.
⚡ Performance Element
Responsible for selecting external action.
💡 Problem Generator
Suggests actions that will lead to new and informative experiences.
07 4 RULES OF AN AI AGENT
1
An AI agent must have the ability to perceive the environment.
2
The observation must be used to make decisions.
3
Decision should result in an action.
4
The action taken by an AI agent must be rational.
08 STRUCTURE OF INTELLIGENT AGENTS
AGENT = ARCHITECTURE + AGENT PROGRAM
The task of AI is to design an agent program that implements the agent function.
ARCHITECTURE
Machinery or devices consisting of actuators and sensors. Examples: personal computer, car, camera.
AGENT FUNCTION
Maps actions from a certain percept sequence (history of what the agent has perceived). Produced by the agent program running on the architecture.
AGENT PROGRAM
The implementation / execution of the agent function.
09 PEAS REPRESENTATION
What is PEAS?

PEAS is a model used to define an AI/rational agent by grouping its properties into four categories. Performance measure is the objective for the success of an agent's behavior.

LETTER STANDS FOR SELF-DRIVING CAR EXAMPLE
P Performance Measure
Objective for success		 Safety, time, legal drive, comfort
E Environment
Surroundings & context		 Roads, other vehicles, road signs, pedestrian
A Actuators
How agent acts		 Steering, accelerator, brake, signal, horn
S Sensors
How agent perceives		 Camera, GPS, speedometer, odometer, accelerometer, sonar
